# Resident Information Management System (RIMS)

A desktop application built with **Python**, **Tkinter**, and **SQLite** to help barangay staff manage resident records more efficiently.  
Instead of typing addresses manually, residents are linked to their **Purok** (zone), making data entry faster and more organized.

---

## ✨ Features
- **Login & Registration**  
  Secure access for authorized users with account creation and login.

- **Purok Management**  
  Add and select Puroks (zones) before managing residents.  
  Residents are automatically linked to the selected Purok.

- **Resident Management (CRUD)**  
  - Add new residents (Name, Age, Contact)  
  - Update existing records  
  - Delete residents  
  - View residents filtered by Purok

- **SQLite Database**  
  Persistent storage of users, Puroks, and residents.
